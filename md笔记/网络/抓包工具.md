# 抓包工具

## 抓包代理 [anyproxy](http://anyproxy.io/cn/)
>这是一个服务代理工具<br>
不能抓所有的包

### 使用指南
* 安装
```
npm install -g anyproxy
```

* 命令行启动AnyProxy，默认端口号8001
```
anyproxy
```
