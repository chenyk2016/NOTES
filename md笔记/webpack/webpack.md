
# 操作

1. 安装淘宝镜像

	npm install -g cnpm --registry=https://registry.npm.taobao.org

2. 创建文件夹

	md webpack-demo && cd webpack-demo

3. 创建文件

	cd.>filename  

4. 创建项目

	npm init

5. 安装依赖

	npm install [--save][--save-dev] <name>
  
6. 指定配置
